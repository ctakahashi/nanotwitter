git status
	This lets you see what has been modified and what has been added (added in terms of it will be committed when you run commit)
	Red is not added, green is added
git add --all
	Or you can type 'git add file_name' to add a specific file and not all
git commit -m "Message" 
	Try to be specific or at least show what changes were made
git push
	If error try 'git push -u origin master' or 'git pull'
